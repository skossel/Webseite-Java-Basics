Wiederholungsstrukturen

Was sind Wiederholungsstrukturen?

Zu den Wiederholungsstrukturen gehören:

  • For Loop
  • While Loop
  • Do while Loop

Wann sollte welche Schleife verwendet werden?

Normale Schleifen

Wenn man weiss wie oft etwas ausgeführt werden soll, sollte man einen For-Loop verwenden, wenn
man nicht weiss wie oft es ausgeführt werden soll benutzt man entweder einen While-Loop oder eine
oder einen Do-while-Loop. Do-while benutzt man, wenn der Codeblock in der Schleife immer mindestens
einmal ausgeführt werden soll, auch wenn die Bedingung der Schleife nicht zutrifft. While benutzt
man, wenn der Inhalt der Schleife nur ausgeführt werden soll, wenn die Bedingung zutrifft.

Endlosschleifen

Wenn man "true" in die Bedingung der While-Schleife schreibt, hat man eine Endlosschleife

Schleifen abbrechen

Mit "break" kann man eine Schleife sofort beenden. Der Code unter der Schleife wird normal ausgeführt.

Wie werden Schleifen geschrieben?

For-Loop

    
      for (int i = 0; i < 5; i++) {
            doSomething();
        }
In dieser Schleife wird "doSomething" fünfmal aufgerufen.

While-Loop

    
      while (!Objects.equals(string1, string2)){
            doSomething();
        }
In dieser Schleife wir "doSomething" solange aufgerufen, bis "string1" gleich "string2" ist.

Do-while-Loop

    
      do {
            doSomething();
        } while (number1 + number2 != 25);
Hier wird "doSomething" immer mindestens einmal aufgerufen, auch wenn "number1" plus "number2" nicht 25 ergibt.

Endlosschleife

    
      while (true){
            doSomething();
            if (String.valueOf(number1).equals(string1)){
                break;
            }
            doSomething();
        }
Diese Schleife wird erst abgebrochen, wenn "number1" gleich "string1" ist und sonst unendlich oft ausgeführt.
