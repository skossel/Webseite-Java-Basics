Verzweigungen

Was sind Verzweigungen?

Kontrollstrukturen werden benutzt, um den weiteren Verlauf des Codes zu steuern.
In Java gibt es if-else if-else-Verzweigungen und switch-Verzweigungen.

Wann wird welche verzweigung benutzt?

Normalerweise benutzt man if-Verzweigungen. Switch wird für spezielle Fälle benutzt.

Wie funktionieren die verschiedenen Verzweigungen?

If-Verzweigungen

Wenn die Bedingung bei if "true" ist, wird der Codeblock im "if" ausgeführt, sonst wird er
übersprungen. Wenn es eine else if Anweisung gibt, wird als nächstes überprüft, ob diese "true" ist,
wenn nicht wird der Code darin ebenfalls übersprungen. Wenn am Ende else steht, wird der Codeblock
darin nur ausgeführt, wenn alle if und else if Bedingungen davor "false" waren. 
        
            if (userInput > counter) {
                System.out.println("Der userInput ist grösser als der counter");
            } else-if (userInput == counter) {
                System.out.println("Der userInput ist gleich gross wie der counter");
            } else {
                System.out.println("Der userInput ist kleiner als der counter");
            }
In diesem Codeabschnitt wird abgefragt, ob der userInput grösser als der counter ist. Wenn dies
der Fall ist wird "Der userInput ist grösser als der counter" in der Konsole ausgegeben. Falls
der userInput gleich gross wie der counter ist, wird "Der userInput ist gleich gross wie der
counter" ausgegeben. Wenn nicht, wird "Der userInput ist kleiner als der counter" ausgegeben.

Switch-Verzweigungen

Das Schlüsselwort switch kann hierbei etwa mit "für den Fall, dass " übersetzt werden.
Übersetzt man das Schlüsselwort case mit "entspricht", dann ergibt sich für die Schreibweise
switch-case Anweisung folgende Bedeutung: Für den Fall, dass der in Klammern stehende Ausdruck
der Konstante 1 entspricht, so sind die hinter dem Doppelpunkt stehenden Anweisungen bis zur
nächsten break-Anweisung auszuführen. Default kann mit einem else gleichgesetzt werden. 
        
            switch (Ausdruck) {
                case Konstante 1:
                    Anweisung;
                case Konstante 1:
                    Anweisung;
                default:
                    Anweisung;
            }
